# Vim minimum

## Modes
Esc - normal mode
i - insert before cursor
a - insert after cursor
o - new line below
O - new line above

## Editing
ciw - change inner word
dd - delete line
yy - copy/yank line
p - paste below/after
u - undo
Ctrl+r - redo

## Movement
w - next word
b - previous word
0 - start of line
$ - end of line
gg - top of file
G - bottom of file

## Search
/text - search text
n - next match
N - previous match

## Save/quit
:w - save
:q - quit
:wq - save and quit
