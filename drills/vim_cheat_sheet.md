# Vim minimum

## Modes
Esc - normal mode
i - insert before cursor
a - insert after cursor
o - new line below
O - new line above

## Editing
ciw - change inner word
dd - delete line
yy - copy/yank line
p - paste below/after
u - undo
Ctrl+r - redo

## Movement
w - next word
b - previous word
0 - start of line
$ - end of line
gg - top of file
G - bottom of file

## Search
/text - search text
n - next match
N - previous match

## Save/quit
:w - save
:q - quit
:wq - save and quit

## Repeat / indent

.  - repeat last change

>> - indent current line right
<< - indent current line left

V  - visual line mode
>  - indent selected lines right
<  - indent selected lines left
gv - reselect last visual selection


## Visual mode

v  - visual mode character selection
V  - visual line mode
d  - delete selected text
y  - yank/copy selected text
Esc - exit visual mode
